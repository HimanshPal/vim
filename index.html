Apreche's vim configuration

My vim configuration is extremely minimal. It sets proper indenting, highlighting, font, colors, etc. It makes very few other modifications. I use default for almost everything. The only major modification is the addition of NERDTree. It is mostly Python centric, because that is what I use most of the time.

Installation

git clone git://github.com/Apreche/vim.git
mv vim ~/.vim
cd ~/.vim
git submodule init
git submodule update
ln -s ~/.vim/vimrc ~/.vimrc
